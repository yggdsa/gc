# 秘密通道导航研究所
“祖师道：‘道’字门中有三百六十傍门，傍门皆有正果。不知你学那一门哩？”悟空道：“凭尊师意思。弟子倾心听从。”祖师道：“我教你个‘术’字门中之道，如何？”悟空道：“术门之道怎么说？”祖师道：“术字门中，乃是些请仙扶鸾，问卜揲蓍，能知趋吉避凶之理。”悟空道：“似这般可得长生么？”祖师道：“不能！不能！”悟空道：“不学！不学！……”
　　书生空洞的眼神逐渐恢复了光彩，脚步停下，同样挤入人群，仔细聆听。
　　接下来，菩提祖师又说了好几种道法让孙悟空挑选，但孙悟空一听不是长生之道，都是一一摇头拒绝。
　　李念凡能明显感觉到，周围的听众呼吸俱是变得急促起来，有些人甚至眼都红了。
　　其中，尤以白无尘四人为甚。
　　他们在心中疾呼，“这可都是无上大道啊！这傻猴子居然都拒绝了！傻猴，傻猴啊！随便学一样本事，那都是堪比仙人的手段，不，是比仙人还要厉害啊！”
　　同时，他们又无比的好奇，这菩提祖师究竟是什么人，居然能教这么多大道，太恐怖了！
　　李公子连这种事情都能详细的讲出来，那他又是什么境界？
　　白无尘不敢再想下去，小心脏受不了。
　　当菩提祖师不想再教愤怒离去时，众人的心又是跟着提起，似乎是自己错失了这无上大道。
　　不过，当孙悟空夜半三更找上祖师时，所有人的呼吸又再度急促起来。
　　李念凡充分的调动起大家的胃口，微微一笑道：“好了，今天就先讲到这里，预知后事如何且听下回分解。”
　　“别啊，李公子求求你再讲会吧。”
　　“哎呀，怎么都一个多时辰了，我娘子恐怕要发飙了！”
　　“李公子，明天还讲吗？我一定来。”
　　所有人都是期待的看着李念凡，就连囡囡的眼神也出现了波动。
　　李念凡揉了揉囡囡的头道：“大家放心，明天这个时间我还会来的。”
　　“李公子，谢谢您，谢谢您！”张大娘也注意到囡囡的变化，喜极而泣，不断的对李念凡鞠躬道谢。
　　“张大娘，你不用这样，等囡囡彻底好了再谢我不迟。”李念凡开口道。
　　白无尘四人走到李念凡面前，如同小学生见老师一般，拘谨到不行。
　　他们的内心同样无比的好奇，孙悟空真的学到长生之道了吗？真正的长生之道究竟是什么？
　　作为修仙者，这是他们的执念。
　　不过，面上却不敢有一点催促。
　　能听到李公子讲这种故事，本身就已经是天大的机缘了。
　　谁能想到，李公子讲这些居然是为了救治一个凡人小女孩，这才是真正的大佬啊。
　　白无尘期待的看着李念凡，小心翼翼的问道：“李公子，明天我们还可以来这里听故事吗？”
　　他有些担忧，这极可能讲的就是某位仙界大佬的故事，真的是他这个小小的修仙者有资格听的吗。
　　“一个故事而已，有什么不可以。”李念凡笑着道。
　　这些修仙者规矩还真多，听故事都要来请示一下，友善，太友善了。
　　白无尘四人大喜，有了李公子作保，他们心中顿时就有底了。
　　白无尘躬身道：“李公子，今日多有打扰，告辞了。”
